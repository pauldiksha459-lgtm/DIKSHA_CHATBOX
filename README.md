# 🤖 Simple Chatbot in Python

## 📌 Description
This project is a simple Python chatbot that interacts with users using predefined responses. It detects keywords in user input and replies accordingly. If no match is found, it gives a default response. It uses randomization for varied replies and demonstrates basic Python concepts like loops, functions, and dictionaries.

The chatbot also uses randomization to make conversations feel more natural by selecting different responses for the same input.

---

## 🚀 Features
- 💬 Interactive conversation with user  
- 🔍 Keyword-based response system  
- 🎲 Random responses for variation  
- ❌ Exit option to end chat  
- 🤖 Beginner-friendly logic  

---

## 🛠️ Technologies Used
- Python  
- `random` module
